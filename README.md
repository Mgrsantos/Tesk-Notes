## 🚀 Sobre o projeto
Aplicação de bloco de notas inspirada no Google Keep. Permite criar, editar e organizar notas em pastas coloridas personalizadas, fixar notas, arquivar, mover para lixeira e pesquisar em tempo real. Dados salvos no localStorage.
